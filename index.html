Boilerplates

I built this for two reasons.

  1. I wanted to start using Github and need a something to host.
  2. I wanted a reason to use jQuery Templates, because I think they're cool.

If you have a boilerplate to add to the list, fork it - add it - submit a pull req.

You can also clone the project with Git by running: 

$ git clone git://github.com/shaundunne/Boilerplates

get the source code on GitHub : shaundunne/Boilerplates
